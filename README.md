# How to Contribute

Create a Database named "PLMPS6G5" in SSMS. Then click on File -> Open -> File and select the schema sql. 

Make sure you are currently working on the correct database by checking at drop-down menu on the top-left of the screen in SSMS **(It should say PLMPS6G5 and not master)**

Click on execute to create the schema, then do the same steps for the sample data found in the main branch of the repository.



Once you've created the database schema and filled the sample data, you can now clone the project and create a new branch named after the feature you want to work with. Use Github Desktop to work with the repository, you can download it [here](https://desktop.github.com/download/)
